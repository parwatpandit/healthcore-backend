HealthCore вЂ” Hospital Management System
What It Is
A full production grade hospital management system that handles everything a real hospital needs digitally. Patients, doctors, appointments, prescriptions, lab results, billing and emergency alerts all in one system.

Real World Use Case
NHS, Bupa, Babylon Health, private clinics, GP surgeries all need exactly this kind of system. This is not a toy project вЂ” this mirrors what real health tech companies in London actually build and maintain.

Complete Tech Stack
Backend

Language: Python
Framework: Django
API: Django REST Framework
Authentication: JWT tokens + Django auth
Password hashing: Django built in with Argon2
Task queue: Celery with Redis
PDF generation: WeasyPrint
Email: Django email with SMTP

Frontend

Framework: React
Language: TypeScript
Styling: Tailwind CSS
Charts: Recharts
HTTP client: Axios
State management: React useState and useContext

Database

Primary: PostgreSQL
Cache: Redis
ORM: Django ORM

DevOps

Containerisation: Docker
Orchestration: Docker Compose
Cloud: AWS EC2 or Railway
Storage: AWS S3 for lab result files

Testing

Backend: Pytest
Frontend: React Testing Library
API testing: Postman
Browser automation: Selenium

Database Models
User Model

id, username, email, password, role (patient/doctor/admin), is_active, created_at

Patient Model

id, user, date_of_birth, blood_type, allergies, emergency_contact, medical_history, created_at

Doctor Model

id, user, specialisation, license_number, years_experience, available_days, created_at

Appointment Model

id, patient, doctor, date, time, status, reason, notes, created_at

Prescription Model

id, patient, doctor, appointment, medication, dosage, duration, instructions, created_at

LabResult Model

id, patient, doctor, test_name, result, status, notes, file, created_at

Bill Model

id, patient, appointment, amount, status, insurance_claim, created_at
